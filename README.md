# DIO Lab Open Source

![Status](https://img.shields.io/badge/status-estudo-blue) ![Stack](https://img.shields.io/badge/stack-Git%20%2B%20GitHub-lightgrey)

## Descrição
Fork/repositório de estudo do laboratório da Digital Innovation One sobre contribuição em projetos open source no GitHub.

## Objetivo
Praticar fluxo de contribuição open source, uso de Git/GitHub, criação de perfil na pasta `community/` e abertura de pull requests.

## Tecnologias utilizadas
- Git
- GitHub
- Markdown
- GitHub Actions

## Funcionalidades
- Materiais de apoio para o lab da DIO.
- Pasta `community/` com perfis de participantes.
- Templates de issues e pull requests.
- Workflow de validação e merge.

## Estrutura de pastas
```text
.
├── .github/         # Templates e workflows
├── community/       # Perfis da comunidade
├── CONTRIBUTING.md  # Guia de contribuição
├── README.md        # Documentação principal
└── .gitignore
```

## Como rodar o projeto
Este repositório é documental. Para contribuir:

```bash
git clone https://github.com/Rafael477/dio-lab-open-source.git
cd dio-lab-open-source
```

Edite ou adicione arquivos Markdown conforme as instruções do lab.
